WSDOT Custom Scripting for Hyperion

wsdot_ResetFile Function

Makes a Hyperion query (.bqy) file smaller for publication on the web by removing data from Results sections and certain Dashboard controls.
[ActiveDocument.]wsdot_ResetFile ([ExclusionList])

Arguments

ExclusionList
Optional. An array of string values. Each value is the name of a Section or Dashboard Control to exclude from the purge.



Return Value

The wsdot_ResetFile function returns true.


Remarks

The following examples use wsdot_ResetFile to clear the contents of all Query sections and Dashboard Controls in the ActiveDocument except those named Q-Subpgm or lstSubpgm. 

var arrExclude = new Array("Q-Subpgm", "lstSubpgm");
wsdot_ResetFile(arrExclude);
//	...or...
wsdot_ResetFile(["Q-Subpgm", "lstSubpgm"]);


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
