WSDOT Custom Scripting for Hyperion

wsdot_ControlExists Function

Determines whether a given control exists on a given Dashboard.
[ActiveDocument.]wsdot_ControlExists (Dashboard, Control)

Arguments

Dashboard
Required. A string value containing the name of the Dashboard that contains the control for which to search.


Control
Required. A string value containing the name of the control for which to search.



Return Value

The wsdot_ControlExists function returns the following values:
If wsdot_ControlExists returns
Dashboard is not the name of a Dashboard object. false
...and "Dashboard is not the name of a Dashboard object." is written to the console.
Control is not the name of a control in Dashboard. false
...and "Control is not the name of a control in Dashboard." is written to the console.
Control is the name of a control in Dashboard. true


Remarks

The following examples use wsdot_ControlExists to determine whether a specifit ListBox exists in a dashboard named "Dashboard": 

//	output whether lstProgram exists
Alert(wsdot_ControlExists("Dashboard", "lstProgram"));


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
