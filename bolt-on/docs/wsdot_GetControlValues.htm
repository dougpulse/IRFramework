WSDOT Custom Scripting for Hyperion

wsdot_GetControlValues Function

Collects values from a given control.
[ActiveDocument.]wsdot_GetControlValues (Dashboard, ControlName)

Arguments

Dashboard
Required. A string value containing the name of the Dashboard that contains the Control containing to evaluate.


ControlName
Required. A string value containing the name of the Control or RadioButton group to evaluate.



Return Value

The wsdot_GetControlValues function returns the following values:
If wsdot_GetControlValues returns
Dashboard is not the name of a Dashboard object. "Dashboard is not the name of a Dashboard object."
ControlName is not the name of a Control in Dashboard. "ControlName is not the name of a control in Dashboard."
ControlName is the name of a control other than a DropDown, ListBox, or TextBox, RadioButton, RadioButton group, or CheckBox. "ControlName is the name of a control that is of a type that is not evaluated.
The control is a DropDown The string value of the selected item.
The control is a ListBox A string containing the space-delimited concatenation of the values of all of the selected items.
The control is a TextBox The Text property of the control.
The control is a RadioButton If Checked == true, the Text property of the control.
If Checked == false, a zero-length string ("").
ControlName is the value of the Group property of a RadioButton. The name of the RadioButton that is selected.
If none are selected, a zero-length string ("").
The control is a CheckBox If Checked == true, the Text property of the control.
If Checked == false, a zero-length string ("").


Remarks

The following examples use wsdot_GetControlValues to determine whether a value is selected in a ListBox: 

Alert(wsdot_GetControlValues("Dashboard", "lstProgram"));


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
