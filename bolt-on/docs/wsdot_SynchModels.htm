WSDOT Custom Scripting for Hyperion

wsdot_SynchModels Function

Synchronizes all data models in the ActiveDocument with their metadata.
[ActiveDocument.]wsdot_SynchModels ()

Return Value

None.

Remarks

The following example uses wsdot_SynchModels to synchronize all data models in the ActiveDocument with their metadata: 

ActiveDocument.wsdot_SynchModels();


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
