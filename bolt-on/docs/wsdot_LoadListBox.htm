WSDOT Custom Scripting for Hyperion

wsdot_LoadListBox Function

Loads information from a Results or Table column into a ListBox or DropDown control.
[ActiveDocument.]wsdot_LoadListBox (DashboardName, ControlName, ResultListName, SourceColName[, FirstItem[, SortDescending[, SortColumn]]])

Arguments

DashboardName
Required. A string value containing the name of the Dashboard that contains the Control to load.


ControlName
Required. A string value containing the name of the Control to load. The control must be a ListBox or DropDown.


ResultListName
Required. A string value containing the name of the Result or Table section to load from.


SourceColName
Required. A string value containing the name of the column in the Result or Table section to load from.


FirstItem
Optional. A string value to place at the beginning of the list. If this value is omitted or a zero-length string (""), no additional item is added to the list.

SortDescending
Optional. If true, the list is sorted in descending order. If false or omitted, the list is sorted in ascending order.


SortColumn
Optional. A string value containing the name of the column in the Result or Table section that will be used for sorting. If omitted, SourceColName is used.



Return Value

The wsdot_LoadListBox function returns the following values:
If wsdot_LoadListBox returns
DashboardName is not the name of a Dashboard object. "DashboardName is not the name of a Dashboard object."
ControlName is not the name of a ListBox or DropDown control in DashboardName "ControlName is not the name of a ListBox or DropDown control in DashboardName"
ResultListName is not the name of a Results or Table section in the ActiveDocument "This document does not contain a Result or Table named ResultListName."
SourceColName is not the name of a column in the Results or Table section named ResultListName "ResultListName does not contain a column named SourceColName."
SortColumn is provided and is not the name of a column in the Results or Table section named ResultListName "ResultListName does not contain a column named SortColumn."
All input is valid "Success"


Remarks

The following examples use wsdot_LoadListBox to populate a listbox control from a Results section: 

//	populate a DropDown from Results
ActiveDocument.wsdot_LoadListBox("Dashboard", "drpOrgCode", "R-Organization", "Organization Code");

//	populate a ListBox from Results, add a first item, and sort ascending by an alternate column.
wsdot_LoadListBox("Dashboard", "lstSubpgm", "R-Subpgm", "Sub Program Name", "*** ALL ***", false, "Sub Program Code");


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
