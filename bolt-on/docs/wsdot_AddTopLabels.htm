WSDOT Custom Scripting for Hyperion

wsdot_AddTopLabels Function

Adds an Item to serve as top labels for a Pivot.
[ActiveDocument.]wsdot_AddTopLabels (PivotSection, Item[, TotalColumn[, InitializeColumnTotals]])

Arguments

PivotSection
Required. A string value containing the name of the Pivot to modify.


Item
Required. A string value containing the name of the Item to add as a top label.


TotalColumn
Optional. A boolean value indicating whether or not to add a total row. Default is false.


InitializeColumnTotals
Optional. A boolean value indicating whether or not to remove all total rows before adding a total row. Default is true.



Return Value

The wsdot_AddTopLabels function returns the following values:
If wsdot_AddTopLabels returns
PivotSection is not the name of a Pivot object "PivotSection is not the name of a Pivot object."
Item is already a top label of the Pivot object "Item is already a top label of PivotSection."
Item is not the name of an Item in PivotSection "Item is not the name of an Item in PivotSection."
All input is valid "Success"


Remarks

The following example uses wsdot_AddTopLabels to add an Item to a Pivot as a top label: 

ActiveDocument.wsdot_AddTopLabels("P-ContractExpenditures", "Contract Number", true);


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
