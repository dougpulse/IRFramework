WSDOT Custom Scripting for Hyperion

wsdot_SetLimitFromListbox Function

Creates a filter for a Query or Results section based on selected values from a ListBox or DropDown control.
[ActiveDocument.]wsdot_SetLimitFromListbox (DashboardName, ControlName, SectionName, ItemName[, TopicName])

Arguments

DashboardName
Required. A string value containing the name of the Dashboard that contains the control containing the values to use.


ControlName
Required. A string value containing the name of the ListBox or DropDown control containing the values to use.


SectionName
Required. A string value containing the name of the Query or Results section to create the filter for.


ItemName
Required. A string value containing the name of the Item (field, column) to filter on.

TopicName
Required if SectionName is a Query section. Ignored if SectionName is a Results section. A string value containing the name of the Topic (table) containing the Item to filter on.



Return Value

The wsdot_SetLimitFromListbox function returns the following values:
If wsdot_SetLimitFromListbox returns
DashboardName is not the name of a Dashboard object "DashboardName is not the name of a Dashboard object."
ControlName is not the name of a ListBox or DropDown control in DashboardName "ControlName is not the name of a ListBox or DropDown control in DashboardName."
SectionName is not the name of a Query or Results section in the ActiveDocument "SectionName is not the name of a Query or Results section in the ActiveDocument."
ItemName is not the name of a column in SectionName "ItemName is not the name of a column in SectionName."
SectionName is the name of a Query section and TopicName is not the name of a Topic in SectionName "TopicName is not the name of a Topic in SectionName."
All input is valid "Success"


Remarks

The following examples use wsdot_SetLimitFromListbox to set a filter on a Query section based on selections made in a ListBox control: 

ActiveDocument.Sections["Q-EX-031 Org Code"].Limits["Sub Program"].Remove();
wsdot_SetLimitFromListbox("Dashboard", "lstSubpgm", 
		"Q-EX-031 Org Code", "Sub Program", "Activity");

ActiveDocument.Sections["Q-EX-031 Org Code"].Limits["Responsible Organization"].Remove();
wsdot_SetLimitFromListbox("Dashboard", "lstRespOrg", 
		"Q-EX-031 Org Code", "Responsible Organization", "Work Order");
var iLimits = ActiveDocument.Sections["Q-EX-031 Org Code"].Limits.Count;
ActiveDocument.Sections["Q-EX-031 Org Code"].Limits.Item(iLimits).Operator
		= bqLimitOperatorBeginsWith;


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
