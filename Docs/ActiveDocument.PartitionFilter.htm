PartitionFilter Function

Creates a PartitionFilter object.
[ActiveDocument.]PartitionFilter ([QueryToFilter[, FilterItem[, PartitionValuesResult][, PartitionValuesColumn]]])

Arguments

QueryToFilter
Opional. The Query object to filter.


FilterItem
Opional. The Limit object (in QueryToFilter) for the partition field.


PartitionValuesResult
Opional. The Result object used to determine the values to add to FilterItem.


PartitionValuesColumn
Opional. The Column object (in PartitionValuesResult) for the partition field.



Return Value

The PartitionFilter function returns a PartitionFilter object.



Remarks

PartitionFilter Example

Requirements

ActiveDocument.loadObjectMethods() must be called before creating a PartitionFilter object or calling any of its methods.
