WSDOT Custom Scripting for Hyperion

wsdot_SetLimitFromResults Function

Creates a filter for a Query or Results section based on selected values from a ListBox or DropDown control.
[ActiveDocument.]wsdot_SetLimitFromResult (ResultsName, ResultsColumnName, SectionName, ItemName, TopicName)

Arguments

ResultsName
Required. A string value containing the name of the Results section from which to get the filter values.

ResultsColumnName
Required. A string value containing the name of the Item (column) containing the values to use to populate the filter. This argument is not used if the section being filtered is a Results section.


SectionName
Required. A string value containing the name of the Query or Results section to create the filter for.


ItemName
Required. A string value containing the name of the Item (field, column) to filter on.


TopicName
Required. A string value containing the name of the Topic (table) containing the Item to filter on. This argument is not used if the section being filtered is a Results section.



Return Value

The wsdot_SetLimitFromResults function returns the following values:
If wsdot_SetLimitFromResults returns
ResultsName is not the name of a Results object "Results name not valid."
ResultsColumnName is not the name of an Item in ResultsName "Results column name not valid."
SectionName is not the name of a Query or Results section in the ActiveDocument "Section name not valid."
ItemName is not the name of a column in SectionName "Column name not valid."
SectionName is the name of a Query section and TopicName is not the name of a Topic in SectionName "Table name not valid."
All input is valid "Success"


Remarks

The following example uses wsdot_SetLimitFromResults to set a filter on a Query section based on values in a Result section: 

//	The Q-Contracts query returns a list of contract numbers.
ActiveDocument.Sections["Q-Contracts"].Process();
wsdot_SetLimitFromResult("R-Contracts", "Contract Number", 
		"Q-Contract Milestones", "Contract Number", "Contract");
ActiveDocument.Sections["Q-Contract Milestones"].Process();
ActiveDocument.Sections["Contract Milestones"].Activate();


Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
