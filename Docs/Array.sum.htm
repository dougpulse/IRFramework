Array.sum Method

Sums the values in an array.
object.sum ()

Arguments

object
Required. Always the name of an Array object.



Return Value

The Array.sum method returns a number or string containing the sum of the array's elements' values.
In the case of a string array, the values are concatenated in order.
If the values in the array are not all of the same type, or if they are not numbers or strings, the return value is undefined.

Remarks

The following example illustrates the use of the Array.sum method. 

var a = new Array("He", "ll", "o", ", Wo", "rld", "!");
Console.Writeln(a.sum());                   //    returns "Hello, World!"

var b = new Array(1, 4, 6, 2, 4);
Console.Writeln(b.sum().toString());        //    returns "17"
