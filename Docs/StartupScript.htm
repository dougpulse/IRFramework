IR Framework

Startup Script

Include this script in the Document.OnStartup event to use the IR Framework in a bqy file.
Copy Code
if (Application.Type != bqAppTypeDesktopClient) {
	//	Turn off the toolbars for more screen acreage for reporting
	for (i = 1; i <= Toolbars.Count; i++) Toolbars.Item(i).Visible = false;
}
//	Load the scripts
eval( (new JOOLEObject("Scripting.FileSystemObject")).OpenTextFile(-2, 1, "\\\\servername\\foldername\\Script\\IRFramework.js").ReadAll() );
