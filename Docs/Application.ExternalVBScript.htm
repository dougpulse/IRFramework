ActiveDocument.ExternalVBScript Function

Creates an ExternalVBScript object.
[ActiveDocument.]ExternalVBScript ([Path[, File[, Script[, Args[, Timeout[, Debug]]]]]])

Arguments

Path
Optional. The path to the file into which we will write the script. If omitted or a zero-length string, the user's temp folder is used.


File
Optional. The name of the file into which we will write the script. If omitted or a zero-length string, "TempScript.vbs" is used.


Script
Optional. The script code to write and run. The default script produces an alert.


Args
Optional. An array of name-value pairs of arguments to pass to the script.


Timeout
Optional. Maximum number of seconds to wait for the script to complete.


Debug
Optional. Set to true to write debug information to the console.

Return Value

The ActiveDocument.ExternalVBScript function returns a ExternalVBScript object that can be used to store and retrieve key-item pairs.


Remarks

The external script must return a value into the pipe file, even if it just indicates success or failure.

If you expect to use the results from the Run method, set the timeout long enough to allow your VBScript to complete. To kick off a long-running script for which no return value is neccessary, set the timeout to 0.<\p>

To keep the Run method from completing before the script has completed its work, have the script create a file named runningMSG in the folder where the script gets written.

The script must write output to a file named pipe.txt in the folder where the script gets written.<\p>

The following example uses ActiveDocument.ExternalVBScript to create an ExternalVBScript object: 

//	Create the array to store the arguments.
var arr = new Array();
//	Enter the arguments in name-value pairs.
arr.push("Filter");
arr.push("Text Files|*.txt|Excel Files|*.xls|All Files|*.*");
arr.push("initdir");
arr.push("C:\\AAWork");

//	Create an ExternalVBScript object.
var o = new ActiveDocument.ExternalVBScript(g_strUserTempPath, "OpenFileDialog.vbs", strOFDlgVBS, arr, 60, true);

//	Write the script to a file.
o.WriteScript();

//	Run the script and get the results.
var sFile = o.RunScript();

//	Use the results.
var sContent = ReadTextFromFile(sFile);

//	Reset the properties of the ExternalVBScript 
//	object and delete its associated files.
o.Reset();
