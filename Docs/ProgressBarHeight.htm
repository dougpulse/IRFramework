ProgressBar.Height Method

Gets or sets the height of the ProgressBar object.
object.Height ([value])

Arguments

object
Required. Always the name of a ProgressBar object.


value
Optional. The value to which to set the property.



Return Value

The ProgressBar.Height property returns the current number of pixels tall to draw the ProgressBar.



Remarks

When attemting to set the property value, it is good to check the return value to ensure that the value was set.

The following example illustrates the use of the Height property. 

var dash = ActiveDocument.Sections["Dashboard"];
dash.pb0001 = new wsdot_ProgressBar(dash);
with (dash.pb0001) {
	Left(300);
	Top(200);
	var iHeight = Height(32);
	if (iHeight == 32) {
		// success
	}
	Width(160);
	Value(0);
	MaxValue(100);
	Show();
	for (var i = 1; i <= dash.pb0001.MaxValue() / 2; i++) {
		var a = (new Date()).valueOf();
		while ((new Date()).valueOf() - a < 10) {
			DoEvents();
			DoEvents();
		}
		dash.pb0001.Value(i * 2);
	}
}


Applies To:

ProgressBar Object



Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
