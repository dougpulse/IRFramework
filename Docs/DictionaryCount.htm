WSDOT Custom Scripting for Hyperion

Dictionary.Count Property

Returns the number of items in a collection or Dictionary object. Read-only.
object.Count

Arguments

object
Required. Always the name of a Dictionary object.



Remarks

The following example illustrates the use of the Count property. 

function CountDemo()
{
   var a, d, i, s;                 // Create some variables.
   d = new wsdot_Dictionary();
   d.Add ("a", "Athens");          // Add some keys and items.
   d.Add ("b", "Belgrade");
   d.Add ("c", "Cairo");
   a = (new VBArray(d.Keys()));    // Get the keys.
   s = "";
   for (i = 0; i < d.Count; i++)   //Iterate the dictionary.
   {
      s += a.getItem(i) + " - " + d(a.getItem(i)) + "<br>";
   }
   return(s);                      // Return the results.
}


Applies To:

Dictionary Object



Requirements

The WSDOT Custom Scripts from the script repository must be loaded into the document.
