Array Average Methods

Numerical average methods for values in an array.
object.mean ()
object.median ()
object.mode ()
object.range ()

Arguments

object
Required. Always the name of an Array object.




Return Value

These methods returns the value as indicated by the method's name.

If the values in the array are not all numeric, the return value is undefined.

Since there can be multiple modes, Array.mode() returns an array containing the resulting values.



Remarks

All of the values in the array must be numbers.

The following example illustrates the use of the Array Average methods.


var a = new Array("He", "ll", "o", ", Wo", "rld", "!");
Console.Writeln(typeof a.mean());         //  returns "undefined"

var b = new Array(1, 4, 6, 3, 4, 7, 3, -1);
Console.Writeln(b.sum().toString());      //  returns "27"
Console.Writeln(b.mean().toString());     //  returns "3.375"
Console.Writeln(b.median().toString());   //  returns "3.5"
Console.Writeln(b.mode().toString());     //  returns "4,3"
Console.Writeln(b.range().toString());    //  returns "8"
