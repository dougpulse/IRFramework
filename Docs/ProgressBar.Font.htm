ProgressBar.Font Method

Gets or sets the font of the ProgressBar object.
object.Font ([value])

Arguments

object
Required. Always the name of a ProgressBar object.


value
Optional. A Font object.



Return Value

The ProgressBar.Font property returns a copy of the current Font object being used by the ProgressBar.



Remarks

The following example illustrates the use of the Font property. 

var dash = ActiveDocument.Sections["Dashboard"];
dash.pb0001 = new wsdot_ProgressBar(dash);
with (dash.pb0001) {
	Left(300);
	Top(200);
	Height(32);
	Width(160);
	var iValue = Value(0);
	if (iValue == 0) {
		// success
	}
	MaxValue(100);
	Show();
	for (var i = 1; i <= dash.pb0001.MaxValue() / 2; i++) {
		var a = (new Date()).valueOf();
		while ((new Date()).valueOf() - a < 10) {
			DoEvents();
			DoEvents();
		}
		dash.pb0001.Value(i * 2);
	}
}


Applies To:

ProgressBar Object
